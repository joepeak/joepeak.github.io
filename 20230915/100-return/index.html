百倍币的梦想与野望

每一个进入币圈的人都有着一夜暴富的梦想，而每一轮牛熊的更替也给了市场参与者这样的机会，这一轮牛熊也给我们这样的机会。如何找到这样的机会就成了每个参与者需要思考的难题。

基于对大量暴富币的研究，本文提供了一些视角和方法论可以供大家参考。


百倍币的一些观察

1、市值前100币种的替换率

2020年3月15日，加密货币市场前100的币种与2021年11月8日市场巅峰时刻的前100币种相比 替换率为67%，下一轮牛市，大概率也如此 现在前100的项目，下一轮牛市往往需要新的叙事来填充。所以投资要投新不要投旧。

2、项目成立时间

百倍项目中，76%是在18-20年熊市周期中成立，上个周期: 24%是在2017年之前成立。下一个周期需要重点关注22年，23年的项目，老币种不建议再去玩了，除非跌破公/私募非常多，让庄家来拉盘。

3、代币单价

投资者普遍偏爱单价较低的品种（例如shib），尽管加密货币的单价并不能完全反映其价值，但市场心理上还是有对低单价品种的偏好。

市场上不缺 10U战神，10U 不够你吃顿好吃的，也不够你抽包好烟，但是对于 5个零乃至更多零开头的代币，10U可以带给你极大的满足，望着钱包里几亿、甚至几十亿的代币发呆也是一种享受。

4、市值与流通量关系

市值和流通量之间的关系也是一个关键指标，尤其是在寻找潜在的百倍币时 市值相对较小的项目有更大上涨空间。

解锁流通分布在40%-60%区间，是上轮牛百倍币的概率分布多的区域。

对于有着天量解锁的代价一定充满警惕。就在几个小时彰，Blur一月一期的解锁转出，4936万枚BLUR(862万美元) 从解锁合约转出流向交易所，还有 Aptos将于11月12日解锁2000万枚APT，价值约1.03亿美元，占流通供应量的8.5%，据研究当释放流动性或占日均交易量100%以上的流动性往往会影响价格。

5、行业领域

从行业领域来看公链 ，DeFi、跨链互操作、基础设施、Layer2 和 X2E 等较为底层的项目，项目背景和团队实力大的背景、团队实力和充足的资金也是重要因素。

6、投资机构的参与

投中百倍币较多的机构有：Multicoin、Polychain、Alameda、BN、A16Z。

大的投资机构与众多项目方有着或多或少的交集和内幕信息，这些都是我们所没办法获取到的，对于项目要留意其募资情况，尤其是有知名投资机构的情况。


百倍币的机会与可能

百倍币，一个渴望但可能实现的梦想，希望通过学习大家都能找到自己的百倍币。

另外，虽然不可能每个人都能找到百倍币，但通过上面的过滤，10倍币、20倍币还是有很大的可能。
